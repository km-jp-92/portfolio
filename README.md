## サービス概要
サービスURL：https://app.pdfsync.tech/  
  
リアル会議の効率化を目的としたWebアプリです。  
主な機能は、PDFページ同期、チャット、個人メモ機能から構成されます。  
アカウント登録不要で、メールアドレスのみで使用可能です。

## 開発背景
現職の役員会に導入していたペーパーレス会議システムに搭載されていた、「発表者がページをめくると全員の画面が同期する」という機能が非常に便利で、自分でも作ってみたいと考えたことが開発のきっかけです。  
本アプリでは、PDFページ同期に加えて、リアルタイムチャット、メモのAI整形機能などを実装し、リアル会議での議論・情報共有をスムーズにするツールとして設計しました。

## 使用想定場面
リアル会議、イベント、セミナー、勉強会、打ち合わせ、教育・講習系など

## 主な機能
 **認証** 
 - メールに記載のURLから、PDFアップロードと閲覧画面に入るためのパスワードを設定
 - パスワード再設定

 **資料のアップロード、削除** 
 - AWS S3へのダイレクトアップロード（Active Storage）

 **PDFページ同期** 
 - 発表者が資料をページ送りすると、聴講者のページがリアルタイム同期する

 **チャット** 
 - チャット（匿名）
 - 各メッセージに対するいいね機能

 **メモ** 
 - 個人メモ
 - メモの文章をAI（OpenAI）で整形

 **PDF閲覧関連** 
 - PDFページ送り
 - 拡大・縮小
 - フルスクリーン
 - タブレット端末でのスワイプによるページ送り（フルスクリーン時）
 - PDFダウンロード
 - PDFをブラウザで開く機能

## 技術構成
| カテゴリ | 技術 |
| :--- | :--- |
| バックエンド | Ruby 3.3.6 / Ruby on Rails 7.2.3 |
| フロントエンド | Stimulus 3.2.2 / React 19.2.0 |
| CSSフレームワーク | Tailwindcss + daisyUI |
| データベース | PostgreSQL |
| ファイルサーバー | AWS S3 |
| デプロイ | Render |

## 認証 / セキュリティ
- パスワード生成ページURL（トークン）の有効期限設定（1時間）
- パスワードハッシュ化（has_secure_password、bcrypt）
- 資料格納・閲覧ページのパスワード認証、有効期限設定（2週間）
- レート制限（Rack::Attack）
